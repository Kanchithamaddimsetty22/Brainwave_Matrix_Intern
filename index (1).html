📊 Sales Data Analysis

A simple project analyzing commercial store sales using Python

🔍 Project Overview

This project demonstrates how to use Python (Pandas, Matplotlib, and Seaborn) to analyze a small dataset of product sales. Key metrics such as total revenue, profit by category, and top-selling product are included.
